# 📒 Address Book Management System in C

## 📌 Project Overview
The Address Book Management System is a menu-driven console application developed using the C programming language.  

This application allows users to manage contact details efficiently with permanent data storage using file handling (CSV format).

---

## 🚀 Features

- ➕ Add New Contacts  
- 📋 View All Contacts  
- 🔍 Search Contact by Name  
- ✏️ Edit Contact Details  
- ❌ Delete Contact  
- 💾 Save Contacts to File  
- 📂 Load Contacts from File  

---

## 🛠 Technologies Used

- C Programming
- Structures
- File Handling (fopen, fprintf, fscanf)
- Modular Programming (.c and .h files)
- CSV File Storage
